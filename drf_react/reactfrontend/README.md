# react-frontend

## Prerequisites
 패키지 매니져 `npm`의 설치가 필요합니다.
 [npm 설치(node.js ver.12.13.1)](https://nodejs.org/dist/v12.13.1/node-v12.13.1-x64.msi)
 (optional) [yarn 설치(yarn ver.1.12.1)](https://yarnpkg.com/latest.msi)

## Install
 필요한 패키지를 설치해줍니다.
```
npm install axios
npm install @material-ui/core
```

## Run
  react start
```
npm start
```
